Data Clustering and Visualization

1) Incident Clusters

Graph Explanation

X-Axis: Incident Number (unique identifier for incidents).

Y-Axis: Encoded Nature of Incidents (numerical representation of incident types).

Clusters: Each color represents a distinct cluster formed using K-Means clustering. Incidents in the same cluster share similar characteristics, such as the type of incident (e.g., theft, accident).

2) Comparison of Incident Types

Graph Explanation

X-Axis: Incident Types (Nature).

Y-Axis: Count of each incident type.

Description: This bar graph compares the number of occurrences of each incident type. It highlights the most frequent incident types in the dataset.

3) Incident Nature Distribution

Graph Explanation

Description: This pie chart shows the proportion of each incident type relative to the total dataset. Each slice represents an incident type, with its size indicating its percentage of the total.
